# Secure E-Health Records Storage using Blockchain and Cryptography

## Overview
The Secure E-Health Records Storage system is a blockchain-based healthcare data management application designed to provide secure, decentralized, and tamper-resistant storage of medical records. The project focuses on enhancing healthcare data privacy, integrity, and secure access control using blockchain and cryptographic techniques.

The system integrates Ethereum Blockchain, IPFS decentralized storage, and Attribute-Based Encryption (ABE) to ensure secure healthcare record management and transparent data handling.

---

## Features
- Secure decentralized healthcare record storage
- Blockchain-based tamper-proof data management
- Attribute-Based Encryption (ABE) for secure access control
- IPFS integration for decentralized cloud storage
- Smart contract-based healthcare data handling
- Enhanced data privacy and integrity
- Secure role-based access management

---

## Technologies Used
- Python
- Solidity
- Ethereum Blockchain
- IPFS
- Attribute-Based Encryption (ABE)
- Cryptography

---

## System Workflow
1. Upload healthcare records securely
2. Encrypt records using Attribute-Based Encryption
3. Store encrypted records using IPFS decentralized storage
4. Record metadata and access information on blockchain
5. Verify user access permissions through smart contracts
6. Provide secure and transparent healthcare data access
